# gitucli GitHub Repo Setup

This project is currently inside a monorepo under `gitu-cli/`. If you want to host the CLI at:

https://github.com/cmgzone/gitucli

use this folder as the root of that repo.

## Recommended Repo Layout (in cmgzone/gitucli)

- `package.json`
- `package-lock.json`
- `src/`
- `dist/` (generated by build; do not commit)
- `.github/workflows/release.yml` (included in this folder)
- `README.md`

## One-time: create the GitHub repo from this folder

Option A (simple copy):
- Copy everything inside `gitu-cli/` into a new empty repo working directory.
- Commit and push to `cmgzone/gitucli`.

Option B (git subtree split):
- Use `git subtree split` to extract `gitu-cli/` into its own repo history, then push to `cmgzone/gitucli`.

## Release publishing

- Push a tag like `v1.2.3` in the `cmgzone/gitucli` repo.
- GitHub Actions builds and attaches a `notebookllm-gitu-cli-<version>.tgz` to the Release.

Users can install directly:

```bash
npm install -g https://github.com/cmgzone/gitucli/releases/download/v1.2.3/notebookllm-gitu-cli-1.2.3.tgz
```
